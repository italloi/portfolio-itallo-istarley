# Portfólio Pessoal — Ítallo Istarley 🚀

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido como requisito prático para o curso de **Ciências da Computação** (Centro Universitário UNINTER).

O objetivo deste projeto é demonstrar minhas habilidades em desenvolvimento web front-end, focado em performance, design responsivo e manipulação de DOM, utilizando tecnologias web nativas, sem a dependência de frameworks externos.

## 🌐 Link do Site
**[Acessar o Portfólio Online](https://italloi.github.io/portfolio-itallo-istarley/)**

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica e acessibilidade.
* **CSS3:** Estilização avançada, uso de variáveis globais (Custom Properties), Flexbox, CSS Grid e media queries para responsividade.
* **JavaScript (Vanilla):** * Lógica de alternância de temas (Dark/Light mode) com persistência em `localStorage`.
  * Intersection Observer API para animações de scroll e navegação dinâmica.
  * Validação customizada de formulários com feedback em tempo real via modais.

## ✨ Principais Funcionalidades
1. **Design System Customizado:** Tema escuro e claro implementados puramente via CSS/JS.
2. **Single Page Application (SPA) Feel:** Navegação suave entre seções com destaque dinâmico no menu superior baseado na posição do scroll.
3. **Menu Mobile Responsivo:** Estrutura adaptável para dispositivos móveis.
4. **Integração de Projetos:** Exibição de repositórios reais (como o sistema *IAtify* e o desafio *StartDB*).
